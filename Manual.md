# Computer Role
1. Generate a random number between 1–100.
2. Keep number fixed until guessed.

# User Role
1. Enter a number.
2. If too high → try lower.
3. If too low → try higher.
4. If correct → win.

# System Role
1. Compare numbers.
2. Display hints.
3. Count attempts.
4. Handle invalid inputs (letters, empty input).
5. Repeat until correct.
6. Show the correct number at the end.
